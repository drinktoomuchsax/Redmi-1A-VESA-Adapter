# Redmi-1A-VESA-Adapter

This is a 75mm VESA port adapter for Redmi 1A monitor, which using a stupid stand without VESA port.  
这个项目能够帮助你将红米 1A 显示器的自带支架变为 75mm 的 VESA 接口。
